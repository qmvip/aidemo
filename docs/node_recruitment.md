## 三、收益详情

### 3.1 收益计算公式

```
月收益 = 基础收益 + 算力奖励 + 活跃奖励 + 治理分红

基础收益 = 质押量 × (年化收益率 ÷ 12)
算力奖励 = (个人算力 ÷ 全网算力) × 激励池 × 50%
活跃奖励 = 在线时间占比 × 基础收益 × 20%
治理分红 = 参与投票提案数 × 基础收益 × 5%
```

### 3.2 收益示例

**主节点 (质押10万WY)**:
```
基础收益: 100,000 × 18% ÷ 12 = 1,500 WY
算力奖励: 1%算力 × 1000万激励池 × 50% = 50,000 WY
活跃奖励: 99% × 1,500 × 20% = 297 WY
治理分红: 5个提案 × 1,500 × 5% = 375 WY
----------------------------------------
月收益: 52,172 WY ≈ 年化 62%
```

**计算节点 (质押1万WY, 1张4090)**:
```
基础收益: 10,000 × 10% ÷ 12 = 83 WY
算力奖励: 0.1%算力 × 1000万激励池 × 50% = 5,000 WY
活跃奖励: 99% × 83 × 20% = 16 WY
----------------------------------------
月收益: 5,099 WY ≈ 年化 61%
```

## 四、加入流程

### 4.1 步骤一: 准备硬件

```
主节点推荐配置:
- CPU: 8核+
- RAM: 16GB+
- 存储: 1TB SSD
- 带宽: 100Mbps对等
- 系统: Ubuntu 20.04+

计算节点推荐配置:
- GPU: A100/H100/4090 (至少1张)
- CPU: 8核+
- RAM: 32GB+
- 存储: 512GB SSD
- 带宽: 50Mbps+
```

### 4.2 步骤二: 购买WY

```
1. 访问支持的交易所
2. 购买WY代币
3. 提现到个人钱包

最低需求:
- 主节点: 50,000 WY
- 计算节点: 5,000 WY
```

### 4.3 步骤三: 运行节点

```bash
# 安装节点软件
git clone https://github.com/wangyi-ai/node.git
cd node
./install.sh

# 质押WY
./wangyi-cli stake --amount 50000 --type master

# 启动节点
./wangyi-cli start
```

### 4.4 步骤四: 监控收益

```bash
# 查看收益
./wangyi-cli rewards

# 查看算力贡献
./wangyi-cli stats

# 查看排名
./wangyi-cli rank
```

## 六、常见问题

### Q1: 需要24小时开机吗？

```
是的，节点需要保持在线以获得完整收益。
建议使用稳定的服务器或数据中心。
```

### Q2: 可以同时运行主节点和计算节点吗？

```
可以，但需要分别质押:
- 主节点: 50,000 WY
- 计算节点: 5,000 WY
硬件也需分开
```

### Q3: 收益什么时候到账？

```
每月1日结算上月收益，
自动发放到质押钱包地址。
```

### Q4: 怎么解押？

```
1. 发起解押申请
2. 等待21天冷却期
3. 解押完成，WY回到钱包

注意: 冷却期内无收益
```

### Q5: 算力不够还能加入吗？

```
可以，主节点不需要GPU。
计算节点需要GPU，但低配也可获得基础收益。
```
